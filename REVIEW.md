# IS進捗管理ツール レビュー記録

レビュー開始：2026-03-23　／　最終更新：2026-04-14（Vite移行完了・デッドコード検出）

---

## このツールは何をするもの？

営業チームが「どの会社に連絡したか」「次はいつ連絡するか」を管理するツール。
AIが通話メモを読んで次の行動を提案したり、Googleカレンダーの空き時間を自動で探したりできる。

---

## ✅ 対応済み一覧

### 🔴 セキュリティ（P0）― 全件対応済み（2026-04-03）

| 問題 | 対応内容 |
|------|---------|
| ログイン情報がブラウザに平文保存されていた | HttpOnly Cookie に移行。localStorage から完全に除去 |
| CSRFトークン未実装 | SameSite=Strict Cookie に移行。追加対策不要な設計に変更 |
| Gemini APIキーがブラウザに露出していた | サーバー側専用エンドポイント（`/api/ai/analyze`）経由に変更 |
| Zoho の秘密情報がブラウザに渡っていた | サーバー側のみで保持。フロントには「設定済みか否か」のみ返す |
| Zoho Webhook に認証がなかった | トークンによるURL認証を追加（Zoho側のURL更新が別途必要） |
| postMessage の送信先が無制限だった | 自サイトのオリジンのみに限定 |
| リダイレクト先をユーザーが操作できた | サーバー側で自動生成に固定。設定画面は読み取り専用に変更 |
| 暗号化キーのフォールバック値が脆弱だった | 未設定時はサーバー起動を失敗させる設計に変更 |
| セッションの有効期限が30日と長すぎた | 7日に短縮 |

## 👍 良い点

- KPIカードの視認性が高く、数値が一目でわかる
- リード検索がインクリメンタルサーチで使いやすい
- モバイル対応・ボトムナビが直感的
- エラーバウンダリが実装されており予期しないエラーに強い
- AI解析結果（興味度・トークポイント・次回アクション日）が実用的
- 候補日検索の設定が柔軟（曜日・バッファ・除外時間帯）
- Zoho CRM との双方向連携で営業フローの自動化が進んだ
