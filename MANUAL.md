## 1. このツールでできること

**IS進捗管理ツール**は、インサイドセールス（電話やメールで営業するチーム）の仕事をサポートするための管理システムです。

### 主な機能一覧

| 機能 | 説明 |
|------|------|
| リード管理 | 見込み客（お問い合わせをくれた会社）の情報を登録・管理する |
| アクション履歴 | 電話やメールなど、顧客への対応記録を残す |
| AIアシスタント | 通話メモをAIが読んで、次のアクションや送るメールを自動で提案する |
| メールテンプレート | よく使うメール文章を保存しておき、すぐに使い回せる |
| 日程調整 | Googleカレンダーと連携して、空き時間を自動で探す |
| ダッシュボード | 今日やるべきことや、全体の進捗をひと目で確認できる |
| マスター設定 | チームメンバーや媒体（ポータルサイトなど）の設定を管理する |

## 2. ログイン・アカウント登録

### ログイン方法

1. ブラウザ（ChromeやSafariなど）でツールのURLを開く
2. **ユーザーID**（メールアドレス）と**パスワード**を入力する
3. 「ログイン」ボタンをクリックする

> **ポイント：** 5回連続でパスワードを間違えると、セキュリティのため15分間ログインできなくなります。

### 新しいアカウントを作る（招待コードが必要）

1. ログイン画面で「新規登録」を選ぶ
2. 管理者から受け取った**招待コード**（一時的なパスワードのようなもの）を入力する
3. 名前・メールアドレス・パスワードを設定する
4. 「登録」ボタンをクリックする

> **招待コードとは？** 許可された人だけが登録できるように、管理者が発行する使い捨ての番号です。24時間で使えなくなります。

### パスワードを忘れたとき

管理者に連絡してください。管理者がパスワードをリセットできます。

## 3. ダッシュボード（トップ画面）

ログイン後に最初に表示される画面です。営業活動の全体像をひと目で確認できます。

### 表示される情報

#### KPIカード（重要な数字の概要）
> KPIとは「Key Performance Indicator」の略で、目標達成度を測る重要な数値のことです。

| カード名 | 説明 |
|----------|------|
| 総リード数 | 登録されている見込み客の合計数 |
| 商談数 | 商談（打ち合わせ）まで進んだ件数 |
| 受注予測 | 受注になりそうな案件の合計金額の見込み |
| 本日のフォロー数 | 今日対応が必要なリストの件数 |

#### コンバージョン率（転換率）
> コンバージョン率とは、問い合わせが来てから実際に受注になるまでの確率のことです。

媒体（ポータルサイト、HP、電話など）ごとに、どれだけ成果が出ているかが確認できます。

#### 本日のフォローリスト
今日アクション（電話・メールなど）をする予定のリードが自動的にリストアップされます。リード名をクリックすると詳細画面が開きます。

## 4. リード管理（見込み客の管理）

**リード**とは、自社サービスに興味を持っている見込み客（会社・担当者）のことです。

### リード一覧を見る

メニューから「リード」を選ぶと、登録されているリードの一覧が表示されます。

- **検索・絞り込み：** 会社名、担当者名、ステータスなどで絞り込めます
- **並べ替え：** 登録日や次回アクション日などで並べ替えられます

### 新しいリードを登録する

1. 「新規登録」ボタンをクリックする
2. 以下の情報を入力する

| 項目 | 説明 | 必須 |
|------|------|------|
| 会社名 | リードの会社名 | ○ |
| 担当者名 | 連絡先の方の名前 | |
| 電話番号 | 連絡先電話番号 | |
| メールアドレス | 連絡先メール | |
| ステータス | 現在の進捗状況（例：初回コール、商談中、受注など） | ○ |
| 媒体・ソース | どこから問い合わせが来たか（例：ポータルサイト、HP） | |
| 担当IS | 担当するインサイドセールス担当者 | |
| 担当営業 | 担当する営業担当者 | |
| 商談日 | 商談の予定日・実施日 | |
| MQLステータス | マーケティング観点での見込み度（Marketing Qualified Lead）| |

3. 入力後「保存」ボタンをクリックする

### リードを編集する

リスト内のリード名をクリックすると詳細画面が開きます。「編集」ボタンを押して内容を変更し、「保存」で確定します。

### リードを削除する

詳細画面の「削除」ボタンから削除できます。削除したデータは元に戻せないため、ご注意ください。

### ステータス（進捗）について

リードの進捗状況を示します。代表的なステータスの例：

| ステータス | 意味 |
|-----------|------|
| 新規 | まだ最初の連絡をしていない |
| 初回コール済み | 最初の電話をした |
| 商談中 | 打ち合わせが始まっている |
| 受注 | 契約が決まった |
| 失注 | 契約に至らなかった |
| ナーチャリング | 今すぐではないが将来的な見込みあり |

> ステータスの種類は管理者がカスタマイズできます。

## 5. アクション履歴（対応記録）

リードごとに、電話やメールなどの対応記録（アクション）を残します。

### アクションを記録する

1. リード詳細画面を開く
2. 「アクション追加」ボタンをクリックする
3. 以下を入力する

| 項目 | 説明 |
|------|------|
| アクション種別 | 電話 / メール / SMS / その他 |
| 結果 | つながった / 不在 / 折り返し待ち / 留守電 など |
| 対応メモ | 通話の内容や気づいたことを自由に記録 |
| 次回アクション日 | 次にフォローする予定の日付 |

4. 「保存」をクリックする

> **ポイント：** 次回アクション日を設定しておくと、その日にダッシュボードの「本日のフォローリスト」に自動で表示されます。

### 過去のアクション履歴を確認する

リード詳細画面の下部に、過去のアクション一覧が日付の新しい順で表示されます。

## 6. AIアシスタント

**Google Gemini AI**（Googleが開発した人工知能）を活用して、営業活動をサポートします。

> **AIとは？** 大量のデータから学習し、人間のような判断や文章生成ができるコンピュータープログラムです。

### AIアシスタントでできること

- 通話メモを読んで**次のアクション（することとその日時）を自動提案**する
- 見込み客の**興味度（高・中・低）を評価**する
- 次回架電時の**トークポイントを提案**する
- 送るべき**メール文章の下書きを自動生成**する

### 使い方

1. リード詳細画面でアクションを記録し、「対応メモ」に通話内容を入力する
2. 「AIに相談」ボタンをクリックする
3. AIが以下を自動で提案してくれる：
   - **次回アクション日時**（例：「3日後の午前10時が最適です」）
   - **次回のトークポイント**（例：「〇〇の課題について深掘りしましょう」）
   - **メール件名と本文**
4. 内容を確認し、「保存する」ボタンで採用する
5. メール文章は「Gmailの下書きに保存」ボタンで、Gmailに直接送れる

> **注意：** AIの提案はあくまで参考です。内容を確認してから使ってください。

### AIを使うための準備（初回のみ・管理者が設定）

- **Google Gemini APIキー**をマスター設定に登録する必要があります
- APIキーはGoogle AI Studio（https://aistudio.google.com）から取得できます

## 7. メールテンプレート機能

よく使うメール文章を「テンプレート（雛形）」として保存しておき、リードに合わせて素早くメールを作成できます。

### テンプレートの種類

| 種類 | 説明 |
|------|------|
| 通常 | 一般的なフォローメール |
| 日程調整 | 面談の候補日時を提示するメール |
| 受注確認 | 契約内容を確認するメール |

### テンプレートを作成する

1. メニューから「メールテンプレート」を開く
2. 「新規作成」ボタンをクリックする
3. テンプレート名・件名・本文を入力する
4. 「保存」をクリックする

### 変数（自動で埋め込まれる情報）について

テンプレートの中に以下のような「変数」を入れると、リードの情報を自動で埋め込んでくれます。

| 変数 | 埋め込まれる内容 |
|------|----------------|
| `{{company}}` | 会社名 |
| `{{contact}}` | 担当者名 |
| `{{meeting_date}}` | 商談日 |

**例：**
```
{{company}} {{contact}}様

先ほどはお電話ありがとうございました。
```
↓ 自動で変換
```
〇〇株式会社 山田様

先ほどはお電話ありがとうございました。
```

### テンプレートを使ってメールを作成する

1. リード詳細画面で「メール作成」ボタンをクリックする
2. 使いたいテンプレートを選ぶ
3. リードの情報が自動で埋め込まれたプレビューが表示される
4. 必要に応じて内容を修正する
5. 「Gmailの下書きに保存」で完成

## 8. 日程調整（カレンダー連携）

Googleカレンダーと連携して、複数のチームメンバーの空き時間を自動で探します。

### 使い方

1. メール作成画面（または専用の日程調整画面）を開く
2. 以下の条件を設定する：

| 設定項目 | 説明 | 例 |
|----------|------|-----|
| 対象メンバー | カレンダーを確認するチームメンバー | 鈴木・田中 |
| 候補期間 | いつ頃の日程を探すか | 今から2週間以内 |
| 所要時間 | 打ち合わせの長さ | 60分 |
| バッファ時間 | 前後に空けておく時間 | 15分 |
| 除外時間帯 | 提示しない時間帯 | 12:00〜13:00（昼休み） |
| 除外曜日 | 提示しない曜日 | 土・日 |

3. 「空き時間を検索」ボタンをクリックする
4. 候補日時が最大3件表示される
5. 使いたい候補にチェックを入れる
6. 「日程調整メールに挿入」ボタンで、メール本文に自動で追加される

> **注意：** この機能を使うには、管理者がGoogleカレンダーAPIの設定をしている必要があります。

## 9. マスター設定（管理者向け）

チーム全体の基本情報を設定・管理する画面です。**管理者権限**を持つアカウントのみ操作できます。

### 設定できる項目

#### チームメンバー
- 担当ISや担当営業として選択できるメンバーを登録・管理する
- 表示用の色（カラー）を設定できる

#### ポータルサイト・媒体
- リードの流入元（どこから問い合わせが来たか）を管理する
- 掲載費用や課金タイプ（月額・成果報酬など）を設定できる
- 媒体ごとのコスト・獲得単価をダッシュボードで確認できる

#### ステータス
- リードの進捗ステータスをカスタマイズできる
- 表示名・色を自由に設定できる

#### ソース（問い合わせ元）
- ポータル / HP / 電話 / 紹介 など、問い合わせの種類を管理する

#### API設定
- **Google Gemini APIキー：** AIアシスタント機能を使うために必要
- **Gmail OAuth クライアントID：** Gmailの下書き保存機能を使うために必要
- **Google Calendar APIキー：** 日程調整機能を使うために必要

> **APIキーとは？** 外部サービス（GoogleのAIやカレンダーなど）を利用するための「合言葉」のようなものです。

## 10. アカウント管理（管理者向け）

ユーザーアカウントの追加・削除・パスワード管理を行います。

### 新しいユーザーを招待する

1. アカウント管理画面を開く
2. 「招待コードを発行」ボタンをクリックする
3. 表示された招待コードを新しいユーザーに共有する
4. ユーザーは招待コードを使って自分でアカウントを作成する

> **招待コードは24時間で無効になります。** 期限が切れた場合は新しいコードを発行してください。

### パスワードをリセットする

1. アカウント一覧から対象のユーザーを選ぶ
2. 「パスワードリセット」ボタンをクリックする
3. 新しい仮パスワードが表示されるので、ユーザーに伝える

### アカウントを削除する

アカウント一覧から対象ユーザーの「削除」ボタンをクリックします。削除すると元に戻せません。

### 権限（ロール）について

| 権限 | できること |
|------|-----------|
| 管理者（Admin） | 全機能 + マスター設定・アカウント管理 |
| 一般ユーザー | リード管理・アクション記録・AIアシスタント・メール作成 |

## 11. よくある質問・困ったときは

### Q. ログインできない

- パスワードを確認してください
- 5回以上間違えた場合は15分待ってから再試行してください
- それでもログインできない場合は管理者にパスワードリセットを依頼してください

### Q. AIアシスタントが動かない

- 管理者がGemini APIキーを設定しているか確認してください
- インターネット接続を確認してください

### Q. Gmailに下書きが保存されない

- 管理者がGmail OAuth設定をしているか確認してください
- ブラウザのポップアップブロックが有効になっている場合、許可が必要です

### Q. カレンダーの空き時間が表示されない

- 管理者がGoogle Calendar APIを設定しているか確認してください
- マスター設定でカレンダー対象のメンバーIDが正しく登録されているか確認してください

### Q. 登録したデータが消えた

- 保存ボタンを押さずに画面を閉じた可能性があります
- 変更後は必ず「保存」ボタンをクリックしてください

### Q. 招待コードの期限が切れた

- 管理者に新しい招待コードの発行を依頼してください
